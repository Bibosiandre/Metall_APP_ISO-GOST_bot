# Metall_APP_ISO-GOST_bot
TG chatbot for working with GOST/ISO documentation for painting and processing metal structures, based on Python and its libraries. The Qven2.5 AI model is installed locally via Ollama and connected via the API. Upon startup, the program also requires a TG Token to connect the bot.
